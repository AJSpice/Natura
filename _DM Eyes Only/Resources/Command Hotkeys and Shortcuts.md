
**To ignore a formatting type a backslash before the format key**
\*This would normally italicize but there is now a hidden backslash*

**Ctrl + P**
Command Palette

**Ctrl + T**
New tab

**Ctrl + O**
Open menu and search all notes

**Insert a # or multiple to create multiple Heading styles**
# Heading 1

## Heading 2

### Heading 3

#### Heading 4

**You can bookmark notes as well by the menu (3 dots) in the top of the note!**

**Use a # to tag notes for easier lookup**
#testtag 

**Ctrl + B** 
**Bold any text**

**Ctrl + I**
*Italicize any text*

**Surround text with ~**
~~Strike though any text~~

**Surround text with \=**
==Highlight any text==

**Dashes will create bullet lists**
- Item 1
- Item 2

**Numbers create numbered lists**
1. Item A
2. Item B

**Ctrl + L to make a checklist**
- [ ] Checklist 1
- [ ] Checklist 2
- [ ] Checklist 3

**One backtick for inline code**
` this is inline code `

**Three backticks for a code block**
```
This is a code block
I can make multiple lines
And then copy them!
```

**Three lines on an empty line will create a heading line**
